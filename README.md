## 📄 Sobre o Projeto
Este ecossistema consolida a aplicação prática de conceitos avançados de Engenharia de Dados, Aprendizado de Máquina e Business Intelligence[cite: 7]. O objetivo central foi cruzar a análise socioeconômica e geoespacial complexa do mercado imobiliário americano (*California Housing*) com a otimização de fluxos operacionais e conversão em plataformas digitais de e-commerce[cite: 7].

A interface foi inteiramente desenhada utilizando a fonte *Inter*, estruturada em contêineres limpos que priorizam a legibilidade e a rápida absorção de métricas de alta gerência (*scannability*)[cite: 7, 8].

---

## 🛠️ Principais Tecnologias
* **Python 3:** Linguagem base utilizada para todo o pipeline de dados, tratamento estatístico e modelagem matemática[cite: 7].
* **PyScript (WebAssembly):** Engine que permite executar o interpretador Python e rodar os scripts de IA nativamente no navegador do cliente, sem necessidade de servidores dedicados de backend[cite: 7].
* **Pandas:** Framework utilizado no tratamento, limpeza e estruturação das matrizes de dados[cite: 7].
* **HTML5 & CSS3:** Arquitetura estrutural e estilização responsiva focada na experiência do usuário[cite: 7, 8].
* **Prism.js:** Biblioteca JavaScript acoplada para a colorização dinâmica do código-fonte (*syntax highlighting*) exposto no portfólio[cite: 7].
* **GitHub Pages:** Infraestrutura de deploy contínuo utilizada para hospedagem pública do ecossistema.

---

## 📊 Engenharia de Dados: Os 5Vs de Big Data
O pipeline de governança foi projetado respeitando os pilares fundamentais de Big Data[cite: 7]:
* **Volume:** Manipulação computacional de 20.640 registros censitários individuais[cite: 7].
* **Velocidade:** Processamento sob demanda executado diretamente na máquina do usuário[cite: 7].
* **Variedade:** Integração analítica de dados estruturados com amostragem de feedbacks textuais (não estruturados)[cite: 7].
* **Veracidade:** Tratamento estatístico de dados espúrios e normalização de escala via z-score[cite: 7].
* **Valor:** Transposição de dados matemáticos frios em inteligência corporativa para redução de riscos e otimização de vendas[cite: 7].

## 📉 Visualizações e Dashboards Integrados
A página consome e apresenta quatro visões estratégicas essenciais para o negócio[cite: 7]:
1. **Matriz de Correlação Linear:** Mapa de calor que decodifica as variáveis mais influentes na precificação de ativos[cite: 7].
2. **Painel de KPIs Gerenciais:** Cartões consolidados com indicadores críticos de desempenho[cite: 7].
3. **Funil de Vendas:** Mapeamento visual e identificação de gargalos de conversão na etapa de checkout[cite: 7].
4. **Segmentação K-Means:** Agrupamento espacial baseado em perfis socioeconômicos de mercado[cite: 7].
